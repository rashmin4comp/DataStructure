# DataStructure Visualizer

An interactive educational platform to visualize and understand Data Structures and Algorithms.

## Features
- **Interactive Animations**: Step-by-step or automated playback.
- **Algorithm Tracing**: See the logic executed line-by-line.
- **Live Variable Tracking**: Monitor the state of pointers, indices, and queues in real-time.
- **Multiple Topics**: Covers Sorting, Stacks, Queues, Linked Lists, Trees, and Graphs.

## Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

## Getting Started

### 1. Install Dependencies
Navigate to the project root directory and run:
```bash
npm install
```

### 2. Run the Development Server
Start the local server with:
```bash
npm run dev
```

### 3. Access the Application
Open your browser and navigate to:
[http://localhost:5173](http://localhost:5173)

## Project Structure
- `src/routes/`: Contains the logic and UI for each data structure topic.
- `src/app.css`: Global styles and color themes.
- `developmentcontext.md`: Technical details for developers.

## How to use
1. Select a topic from the **Main Menu**.
2. Adjust the **Speed** or **Size** if applicable.
3. Input values where needed (e.g., node value to insert).
4. Use **Play** for continuous animation or **Next Step** to move line-by-line.
5. Watch the **Algorithm** and **Variables** panels to understand the internal logic.
